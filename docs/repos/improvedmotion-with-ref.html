ImprovedMotion (with reference)

Motion-conditioned image and video generation with reference-based modeling.

Research codebase with a StyleGAN-style generator, discriminators, motion and landmark pipelines, and multiple training entry points. Core architecture under networks/; run scripts from the repository root with the project on PYTHONPATH.

Highlights

  • Training loop with TensorBoard and checkpoints under exps/
  • Video datasets (VoxCeleb-style and custom loaders)
  • Alternate paths: 3D training, lip-focused trainers, run_demo.py for inference
  • Cog (predict.py, cog.yaml) for containerized inference

Tech

Python · PyTorch · Perceptual losses (VGG, LPIPS, DISTS, SSIM) · Landmarks / PIPNet
