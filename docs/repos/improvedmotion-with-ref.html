ImprovedMotion (with reference)

Research codebase for motion-conditioned image and video generation: StyleGAN-style generator, discriminators, motion and landmark pipelines, and several training entry points. Run scripts from the repo root with the project on PYTHONPATH.

View on GitHub ← Back to home

Highlights

  • Core — networks/ holds generator, discriminator, motion extractor, SPADE-style blocks, PIPNet / landmark helpers.
  • Training — train.py + trainer.py with perceptual losses (VGG, LPIPS, DISTS, SSIM, etc.) and TensorBoard.
  • Data — VoxCeleb-style and custom video loaders (dataset.py, VideoDataset*.py).
  • Alternate tracks — 3D and lip-focused trainers (Train3d.py, LipTrainer.py, …) for specialized experiments.
  • Shipping — predict.py + cog.yaml for Cog / Replicate-style container inference.

Outputs & checkpoints

Experiments land under exps/ (often gitignored); demos and visuals under res/. Place pretrained weights where predict.py expects (e.g. checkpoints/vox.pt, taichi.pt, ted.pt) per README.
