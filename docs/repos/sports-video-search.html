Sports Video Search

Production-oriented MVP: hybrid semantic + structured search over sports clips.

Combines Vertex AI (Gemini for query parsing and summaries, text embeddings), PostgreSQL for metadata filters, and FAISS for vector retrieval. FastAPI exposes search, health, and readiness endpoints suitable for containers and Cloud Run.

Flow

  1. Query understanding — Gemini extracts structured facets (JSON); falls back gracefully if Vertex is unavailable.
  2. Retrieval — Postgres for filters; FAISS for top‑K similarity; dedupe by clip id.
  3. Ranking — Blend of editorial signals, vector similarity, and facet bonuses.
  4. Answer — Gemini summarizes top hits in neutral, sports-appropriate language.

Tech

Python · FastAPI · Vertex AI · PostgreSQL · FAISS · Google Cloud Run
