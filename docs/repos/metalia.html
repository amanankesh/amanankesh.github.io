METALIA

PyTorch implementation for one-shot facial image animation from a driving video.

Animate a source face image using motion from driving video. Pretrained settings for VoxCeleb, Taichi, and TED-talks configurations (vox, taichi, ted). Includes training scripts, run_demo.py for local inference, and Cog-based predict.py for containerized deployment.

Highlights

  • Generator, encoder, discriminator under networks/
  • Distributed training via train.py / train_w_lmks.py
  • Evaluation helpers and Replicate/Cog workflow

Tech

Python · PyTorch · One-shot animation
