METALIA

PyTorch implementation for one-shot facial image animation: animate a single source face using motion from a driving video. Pretrained configurations align with common benchmarks (VoxCeleb, Taichi, TED-talks style settings).

View on GitHub ← Back to home

Workflow

  1. Prepare Python 3.8+ and PyTorch for your CUDA/CPU setup.
  2. pip install -r requirements.txt after PyTorch.
  3. Download pretrained checkpoints into checkpoints/ as vox.pt, taichi.pt, ted.pt (see repo for exact format).
  4. Run run_demo.py for local image + video → MP4; use predict.py for Cog builds.

Key paths

  • networks/ — generator, encoder, discriminator.
  • train.py / train_w_lmks.py — distributed training entry points.
  • run_demo.py — default output folder res/.
  • evaluation.py — metrics (may require lpips and related deps).
  • cog.yaml — CUDA 11 + Python 3.8 baseline for Replicate/Cog.
