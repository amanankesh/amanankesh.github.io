Cricket Info Motion Graphics

Modular cricket video pipeline: raw footage → chalkboard / motion-graphics style output.

A Python pipeline that turns match footage into a stylized look: edge-based illustration, emphasis on players and ball, static overlay removal, and a final cinematic grade. Outputs include per-frame masks, cleaned video, edge-map video, composites, and a graded final.mp4.

Pipeline (high level)

Static removal → Player segmentation (YOLO + SAM2) → Ball segmentation → Edge map (Canny + chalk BG)
→ Highlighting composite → Film-style effects

Tech

  • Python orchestration (main.py)
  • YOLO prompts + SAM2 video propagation for player/ball masks
  • Temporal static logo/banner handling
  • Edge maps, compositing, post (grain, flicker, grade)
