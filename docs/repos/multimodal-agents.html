multimodal-agents

Monorepo for LLM routing, TV analytics, and GPU inference on Kubernetes.

Multiple components with their own install story: Google Gemini and Azure OpenAI–compatible routers, MCP server for TV viewership KPIs (BigQuery), Helm charts for Ollama and vLLM on Kubernetes, and supporting scripts. Each top-level directory has its own README and dependencies.

Components (examples)

  • gemini_router — Gemini router with OpenAI-compatible API and artifact generation
  • gptrouter — Azure OpenAI proxy
  • mcp-server — FastAPI + BigQuery for TV KPIs
  • vllm-helm-charts / ollama-helm — Kubernetes deployment
  • priority-paygo-poc — Vertex AI experiments

Tech

Python · FastAPI · Kubernetes · Helm · Vertex AI · vLLM · Ollama · BigQuery
