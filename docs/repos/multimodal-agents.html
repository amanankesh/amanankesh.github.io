multimodal-agents

A monorepo of production-adjacent tooling: OpenAI-compatible routers for Gemini and Azure GPT, TV viewership analytics over BigQuery, and Helm-centric GPU inference (vLLM, Ollama) on Kubernetes. There is no single install for the whole tree—each folder has its own requirements or chart.

View on GitHub ← Back to home

Top-level components

Directory Role
gemini_router/ Google Gemini behind an OpenAI-compatible API; artifact generation hooks.
gptrouter/ Azure OpenAI (e.g. GPT‑5 class models) / image proxy, OpenAI-compatible.
mcp-server/ FastAPI service exposing TV viewership KPIs via BigQuery.
mcp-server-helm/ Helm chart for the MCP server deployment.
ollama-helm/ Helm chart for Ollama on Kubernetes.
vllm-helm-charts/ vLLM stack: router, workers, Ray, LoRA, cache patterns.
vllm-helm-ps/ Extra GKE-oriented snippets (PVC, HPA, model config).
priority-paygo-poc/ Vertex AI priority / pay-as-you-go experiment.
scripts/, filestore/ Utilities and shared storage manifests for GPU workloads.

Operations notes

Root files include GPU operator notes, a CUDA vector-add job for verifying GPUs on cluster, and environment templates. Treat each subdirectory as its own service with its own README.
